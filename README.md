# CS3502 Project 1
This project involves working with threads in C to implement various synchronization mechanisms. The project is divided into four phases, each building upon the previous one to demonstrate different aspects of thread synchronization. The First phase focuses on creating and managing threads, while subsequent phases introduce mutexes, condition variables, and semaphores to handle synchronization challenges.

# Compilation Instructions
Before executing the following commands, ensure you are in the correct directory.
Each phase can be compiled with the following commands:
### Phase 1:
`gcc -Wall -Wextra -pthread phase1.c -o phase1`

### Phase 2:
`gcc -Wall -Wextra -pthread phase2.c -o phase2`

### Phase 3:
`gcc -Wall -Wextra -pthread phase3.c -o phase3`

### Phase 4:
`gcc -Wall -Wextra -pthread phase4.c -o phase4`

# Execution Instructions
Each phase can be executed with the following commands:
### Phase 1:
`./phase1` 

### Phase 2:
`./phase2`

### Phase 3:
`./phase3`

### Phase 4:
`./phase4`
